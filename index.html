Steps for Using Git and GitHub

  1. Initialize a new Git repository: git init
  2. Add files to staging: git add .
  3. Commit changes: git commit -m "Initial commit"
  4. Create a new repository on GitHub.
  5. Connect local repo to GitHub: git remote add origin https://github.com/your-username/your-repository.git
  6. Push changes to GitHub: git push -u origin master
  7. To pull updates from GitHub: git pull origin master
  8. Create a new branch: git checkout -b new-feature
  9. Merge branch into master: git merge new-feature
  10. Push updates: git push origin master
